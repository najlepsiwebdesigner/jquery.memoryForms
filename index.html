memoryForms test page and example page

memoryForms is a simple jQuery plugin with purpose of saving state of every form on the page. Initial setup: 

	<script src="jquery-1.8.3.min.js"></script>
	<script src="json2.js"></script>
	<script src="jstorage.js"></script>
	<script src="jquery.memoryForms.js"></script>
	<script>
		$(function(){
			$('form').memoryForms(
				{TTL:5000}, //keep the information for 5 seconds - only configuration option
				null // directly pass elements to be preserved - accepts collection of jQuery objects			
			); 
		});
	</script>
